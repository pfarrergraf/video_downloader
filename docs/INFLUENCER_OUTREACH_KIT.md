## 1. Grundregeln der Ansprache

Bevor du irgendeine Vorlage abschickst:

1. **Kanal wirklich ansehen.** Mindestens zwei, drei Videos oder Posts des
   Creators anschauen. Wenn du nicht sagen kannst, worum es auf dem Kanal geht,
   schreib nicht.
2. **`{{video_example}}` konkret benennen.** Der Platzhalter steht für ein
   echtes Video, das du gesehen hast — mit Titel oder erkennbarem Inhalt. „Dein
   Content ist super" ist Spam; „dein Video zu {{video_example}}" ist ein
   Gesprächsanfang.
3. **Kein Copy-Paste-Spam.** Jede Nachricht wird vor dem Absenden angepasst:
   Anrede, Bezug, Tonfall. Wenn zwei Creator dieselbe Nachricht wortgleich
   bekommen könnten, ist sie nicht fertig.
4. **Nur öffentliche Kontaktwege (DSGVO-freundlich).** Kontaktiere Creator nur
   über Kanäle, die sie selbst öffentlich für Geschäftsanfragen angeben:
   Business-E-Mail im Profil/Impressum, offene DMs, Kontaktformular. Keine
   privaten Adressen recherchieren, keine gekauften Kontaktlisten, keine
   automatisierten Massen-DMs.
5. **Ehrlich bleiben.** Sag von dir aus, dass die Auszahlung im Partnerprogramm
   noch nicht freigeschaltet ist. Wer sich jetzt registriert, ist früh dabei —
   mehr versprechen wir nicht.
6. **Ein Follow-up, dann Ruhe.** Nach 5–7 Tagen einmal freundlich nachfassen
   (Vorlage 7). Kommt keine Antwort, ist das eine Antwort.
7. **Keine Zahlenfantasien.** Keine Einkommensversprechen, keine erfundenen
   Conversion-Raten, keine „Creator wie du verdienen X". Die einzigen Zahlen,
   die wir nennen, sind die echte Provisionsstaffel und die App-Fakten.

## 13. Tracking der Outreach-Pipeline

Eine einfache Tabelle reicht völlig — als Markdown-Datei, Spreadsheet oder
Notiz. Pro Creator eine Zeile; Status z. B. „kontaktiert", „Follow-up",
„interessiert", „Partner", „abgesagt", „keine Antwort".

| Creator | Kanal | Kontaktweg | Datum Erstkontakt | Status | Follow-up-Datum | Notizen |
|---|---|---|---|---|---|---|
| {{creator_name}} | {{channel_topic}} (z. B. Instagram) | DM / E-Mail | 2026-07-10 | kontaktiert | 2026-07-16 | Bezug: {{video_example}} |
|  |  |  |  |  |  |  |
|  |  |  |  |  |  |  |

Regeln fürs Tracking:

- **Follow-up-Datum sofort eintragen** (Erstkontakt + 5–7 Tage), damit das
  einmalige Nachfassen nicht vergessen wird — und danach kein weiteres.
- **„Keine Antwort" nach dem Follow-up ist ein Endstatus.** Nicht erneut
  anschreiben.
- **Notizen persönlich halten:** Welches Video war der Anlass, was wurde
  versprochen, welche Fragen kamen. Das ist die Grundlage dafür, dass die
  nächste Nachricht wieder persönlich klingt.
- **Datensparsam bleiben:** Nur öffentlich verfügbare Kontaktdaten und das
  Nötigste notieren; Einträge abgesagter Creator nach angemessener Zeit
  löschen.
